Gav & Sam's Wedding invite

Letterpress Project

Working along side Gav, a designer and good friend of mine, we made invites for his up and coming wedding. For this, I yet again was able to use Pinched Post's studio to create an awesome letterpressed wedding invite.

These were done with a mix of wood and lead type with some lead decorative elements added in to give the invite some extra decoration and printed on 150GSM textured paper.

Steadwatch

Fun project

This was a small, fun project I through together when I wanted to learn a bit more about js structure, backbonejs and working with different API's.

The idea came from people posting lyrics and trying to be vauge on their social networks and being able to figure out what they are talking about.

I started this project out very simply as I have plans to use it to learn more about itteration on projects and adding features in chunks instead of doing it all at once.

This was doen using backboneJs and the lyricsnmusic api.

Letterpress Project
